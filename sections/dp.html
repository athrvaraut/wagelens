Algorithm 03 — DP / Memoization

Salary Growth Optimizer

Dynamic Programming solves problems by breaking them into overlapping subproblems and storing results. Here: find the maximum cumulative salary over career years, choosing to stay or jump each year.

⏱ Time: O(n·m)  |  Space: O(n·m) → O(n) with rolling array optimization

Overlapping Subproblems

The best salary at year 4 depends on years 1–3. Without memoization, we'd recompute these repeatedly. DP stores each state in a table — dp[year][choice] — computed only once.

Optimal Substructure

The maximum salary through year N = max(stay_salary_N, jump_salary_N) + best_salary_through_year_{N-1}. The global optimum is built from locally optimal subproblems.

DP State Table — Year × Choice

Recurrence Relation

dp[i][0] = max(dp[i-1][0], dp[i-1][1]) + stay[i]
dp[i][1] = max(dp[i-1][0], dp[i-1][1]) + jump[i]
dp[0][0] = stay[0]  |  dp[0][1] = jump[0]
Answer = max(dp[n-1][0], dp[n-1][1])
